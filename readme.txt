== Description ==

WP Developer can write raw PHP code in WordPress Post or Page by using simple prefix & postfix. 
<br/>This Plugin is totally free & open source. OSS Write Raw PHP plugin has no commercial or premium version. It will open source in future. This plugin is developed by an open source solver team, so you can get any kind of help from them within short time.<br/>
Contact: Open Source Solver, OpenSourceSolver@gmail.com <br/>
FB: http://www.facebook.com/OpenSourceSolver

== Installation ==

1. Download the zip file of OSS Write Raw PHP plugin from WP Plugin directory, then unzip & upload that unzip folder to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. This plugin no need to configure or settings
